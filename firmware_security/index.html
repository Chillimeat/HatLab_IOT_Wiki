    概述¶

    固件安全主要研究的是硬件设备内部的固件，通过对固件中程序的代码逻辑进行逆向分析，从而从中发其中的安全问题。

    目录主要分为以下几部分来归类：

    1. 固件的获取
    2. 固件的处理
    3. 固件的漏洞分析和挖掘
    4. 固件安全实例
