    概述¶

    因为 IOT 设备的固件大多数是基于 Linux 系统，但是也不乏一些实时操作系统，如 VxWorks、FreeRTOS 文件系统等。这些系统是无法提取出例如类似 linux 这样根目录的，而这些系统的一整个二进制文件。
